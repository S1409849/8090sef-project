# HKIA Flight Information Display System (FIDS)

A real-time, interactive web clone of the HKIA LED departure screens. Built with a decoupled architecture using React and Python FastAPI.
## ✨ Features

* **UI/UX:** Simulating real airport LED flip-boards.
* **Auto-Refresh:** Silently polls the backend every 60 seconds to fetch the latest flight statuses without disrupting the user experience.
* **Dynamic Search & Filtering:** Instantly filter flights by flight number, destination, or gate using a debounced search input.
* **Color-Coded Statuses:** 
* * 🔴 **Red:** Cancelled
  * 🟡 **Blinking Yellow:** Boarding Soon / Final Call
  * 🟠 **Orange:** Gate Closed
  * ⚪ **Gray/Muted:** Departed
